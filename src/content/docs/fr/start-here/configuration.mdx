Le fichier de configuration de StudioCMS permet de configurer différents aspects de StudioCMS dans votre projet Astro. Il vous permet de définir des paramètres tels que la configuration de la base de données, les modules d'extension, le registre des composants, la localisation et les fonctionnalités.

<Aside>
Le fichier `studiocms.config.*`, tout comme le fichier de configuration d'Astro, prend en charge les extensions suivantes : `.cjs`, `.js`, `.mjs`, `.cts`, `.ts`, `.mts`. L'exemple suivant illustre l'utilisation de l'extension de fichier `.mjs`.
</Aside>

### Exemple de structure de fichiers

<FileTree>

- .env
- astro.config.mjs
- **studiocms.config.mjs**
- package.json
- src
  - ...

</FileTree>

### Exemples de fichiers de configuration

```ts twoslash title="astro.config.mjs"
import node from "@astrojs/node";
import studioCMS from "studiocms";
import { defineConfig } from "astro/config";

export default defineConfig({
  site: "https://demo.studiocms.dev/",
  output: "server",
  adapter: node({ mode: "standalone" }),
  integrations: [ studioCMS() ],
});
```

```ts twoslash title="studiocms.config.mjs"
import { defineStudioCMSConfig } from "studiocms/config";

export default defineStudioCMSConfig({
  dbStartPage: false,
  // autres options de configuration
});
```
