## Corrections de bugs

- Implémente des correctifs pour la première configuration, les routes d'API pour l'authentification, le CSS pour la police et la configuration du type de page Markdown.

<Aside title="À noter">
Veuillez noter que dans la version `beta.16`, nous avons introduit une modification avec rupture de compatibilité concernant l’algorithme de hachage des mots de passe. Bien que les anciens mots de passe fonctionnent toujours, nous recommandons aux utilisateurs de les mettre à jour après la mise à niveau vers la version `beta.21`. La prise en charge de l’ancien hachage des mots de passe sera supprimée dans une prochaine version.
</Aside>
