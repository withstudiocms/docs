Der Paketkatalog ist eine Sammlung von Paketen von StudioCMS oder seinen Community-Mitgliedern, die nicht standardmäßig in StudioCMS enthalten sind. Alle hier aufgeführten Pakete wurden vom StudioCMS-Team oder seinen Mitwirkenden überprüft und getestet.

Jedes Paket, das als <Badge text="Plugin" variant="tip"/> gekennzeichnet ist, ist ein Paket, das die Funktionalität von StudioCMS oder seinem Dashboard erweitert.

Jedes Paket, das als <Badge text="Publicly Usable" variant="caution"/> gekennzeichnet ist, kann installiert und in Projekten verwendet werden, die nicht auf StudioCMS angewiesen sind. Sie können eigenständig arbeiten.
