La API de almacenamiento de StudioCMS proporciona varios asistentes de JavaScript para facilitar la interacción con los backends de almacenamiento. Estos asistentes simplifican tareas como resolver identificadores y claves de almacenamiento, además de proporcionar definiciones de tipos para una mejor seguridad de tipos y autocompletado de código.

## Asistentes de JavaScript

```ts twoslash title="ejemplo-api-almacenamiento.ts"
import {
    resolveStorageIdentifier,
    resolveStorageKey,
} from 'studiocms/storage-api';

const identificadorAlmacenamiento = await resolveStorageIdentifier('storage-file://my-file.txt', {
    baseUrl: 'https://ejemplo.com/', // Usa el valor de `site` de "astro:config/server"
});

const claveAlmacenamiento = await resolveStorageKey('mi-archivo.txt', {
    baseUrl: 'https://ejemplo.com/', // Usa el valor de `site` de "astro:config/server"
});
```
