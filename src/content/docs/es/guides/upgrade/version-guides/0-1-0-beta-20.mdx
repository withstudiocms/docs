## Correcciones de errores

- Corregidas las rutas de configuración inicial que utilizaban una funcionalidad de middleware no disponible durante la configuración inicial, causando que la configuración inicial falle.

<Aside title="Nota">
Por favor, ten en cuenta que en `beta.16` introdujimos un cambio importante con el algoritmo de hash de contraseñas. Si bien las contraseñas pueden seguir funcionando, te recomendamos que los usuarios actualicen sus contraseñas después de actualizar a `beta.20`. El soporte para el hash de contraseñas heredadas se eliminará en una versión futura.
</Aside>
