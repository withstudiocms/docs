## Correcciones de errores

- Implementadas correcciones para la configuración inicial, las rutas de API de autenticación, el CSS para la fuente y la configuración de la página de tipo Markdown.

<Aside title="Nota">
Por favor, ten en cuenta que en `beta.16` introdujimos un cambio importante con el algoritmo de hash de contraseñas. Si bien las contraseñas pueden seguir funcionando, te recomendamos que los usuarios actualicen sus contraseñas después de actualizar a `beta.21`. El soporte para el hash de contraseñas heredadas se eliminará en una versión futura.
</Aside>
