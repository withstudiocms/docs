El ecosistema StudioCMS es una colección vibrante y en constante crecimiento de herramientas, plugins e integraciones que mejoran la funcionalidad principal de StudioCMS o extienden el framework Astro. Estos paquetes del ecosistema son desarrollados tanto por el equipo de StudioCMS como por la comunidad en general, proporcionando a los usuarios una amplia gama de opciones para personalizar y optimizar su experiencia con StudioCMS.

## Paquetes destacados

### Biblioteca UI de StudioCMS

`@withstudiocms/ui` es una biblioteca de componentes UI completa diseñada específicamente para crear interfaces de usuario dentro de proyectos StudioCMS. Proporciona un amplio rango de componentes preconstruidos y personalizables que siguen los principios de diseño de StudioCMS, facilitando la creación de interfaces consistentes y visualmente atractivas.

<ReadMore>
  Aprende más sobre la Biblioteca UI de StudioCMS en la [Documentación de la Biblioteca UI](/es/ecosystem/packages/studiocms-ui/).
</ReadMore>

### Lenguaje de plantillas

`@withstudiocms/template-lang` es un motor de plantillas ligero y flexible diseñado para renderizar contenido dinámico en proyectos StudioCMS. Soporta interpolación de variables, facilitando la creación de plantillas dinámicas para correos electrónicos, documentos y páginas web.

<ReadMore>
  Aprende más sobre el Lenguaje de plantillas en la [Documentación del Lenguaje de plantillas](/es/ecosystem/packages/template-lang/).
</ReadMore>

### Cliente de base de datos Kysely

`@withstudiocms/kysely` es un generador de consultas SQL seguro para TypeScript, construido sobre la biblioteca Kysely. Proporciona una forma poderosa y flexible de interactuar con bases de datos mientras garantiza la seguridad de tipos y reduce errores de tiempo de ejecución. Este paquete es usado internamente por StudioCMS para operaciones de base de datos pero, también puede ser usado directamente en tus proyectos.

<ReadMore>
  Aprende más sobre el Cliente de base de datos Kysely en la [Documentación del Cliente de base de datos Kysely](/es/ecosystem/packages/kysely/).
</ReadMore>

### CFetch

`@studiocms/cfetch` es una integración de Astro que proporciona una función fetch con caché para proyectos Astro SSR. Mejora la API fetch estándar agregando capacidades de caché, permitiéndole almacenar en caché las respuestas y mejorar el rendimiento para aplicaciones renderizadas en el lado del servidor.

<ReadMore>
  Aprende más sobre CFetch en la [Documentación de CFetch](/es/ecosystem/packages/cfetch/).
</ReadMore>
