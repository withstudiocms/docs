# Introducción

Una colección de herramientas útiles disponibles durante el modo de desarrollo en Astro.

## Instalación

<Tabs syncKey='install-method'>
  <TabItem label="Método Astro" icon='astro'>
<Steps>
1. Instala el paquete y añádelo a tu configuración de Astro usando el siguiente comando:

   <PackageManagers type="exec" pkg="astro" args="add @studiocms/devapps" />

2. Actualiza tu archivo de configuración como se muestra en [Ejemplo de Configuración](#ejemplo-de-configuración) a continuación.
</Steps>
  </TabItem>
  <TabItem label="Método Manual" icon='seti:powershell'>
<Steps>
1. Instala el paquete usando el siguiente comando:

   <PackageManagers pkg="@studiocms/devapps" />

2. Añade `@studiocms/devapps` a tu archivo de configuración de Astro:

   ```ts title="astro.config.mjs" ins={3, 9}
   import { defineConfig } from 'astro/config';
   import studiocms from 'studiocms';
   import devApps from '@studiocms/devapps'; 

   export default defineConfig({
      site: "https://example.com",
      integrations: [
          studiocms(), // REQUERIDO
          devApps(), 
      ],
   });
   ```
</Steps>
  </TabItem>
</Tabs>

## Uso

Todas las herramientas solo estarán disponibles durante `astro dev` y no estarán disponibles durante las implementaciones de producción!

### Ejemplo de configuración

```ts twoslash title="astro.config.mjs" {2, 8} ins={3, 9}
import { defineConfig } from 'astro/config';
import studiocms from 'studiocms';
import devApps from '@studiocms/devapps';

export default defineConfig({
  site: "https://example.com",
  integrations: [
    studiocms(), // REQUERIDO
    devApps(/* Ver Opciones de Configuración a continuación */),
  ],
});
```

### Opciones de Configuración

Lista completa de opciones disponibles para la integración `devApps`, con valores por defecto mostrados. **Todas las opciones son opcionales**.

```ts twoslash 
import devApps from '@studiocms/devapps';
devApps({
// ---cut---
endpoint: '_studiocms-devapps',
verbose: false,
appsConfig: {
  wpImporter: true,
},
// ---cut-after---
})
```

## Herramientas

### Importador de WordPress

Herramienta de desarrollo para importar publicaciones y páginas de WordPress en la base de datos de StudioCMS.

#### Requisitos

- Integración de StudioCMS
- WordPress instalado
- `@studiocms/blog` (Opcional: para importar publicaciones bajo un blog)

- Aplicación de la barra de herramientas
