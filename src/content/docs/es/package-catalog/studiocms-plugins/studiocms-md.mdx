# Introducción

Este plugin habilita el soporte de MD dentro de StudioCMS.

## Instalación

1. Instalar el paquete usando el siguiente comando:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/md' />

2. Tu configuración de StudioCMS ahora debería incluir `@studiocms/md`:

   ```ts twoslash title="studiocms.config.mjs" {2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import md from '@studiocms/md';

    export default defineStudioCMSConfig({
      plugins: [
        md(),
      ],
    });
   ```
