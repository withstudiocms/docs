# Introducción

Este complemento habilita el soporte de MDX dentro de StudioCMS.

<Aside>
Este complemento requiere el complemento `@studiocms/md`.
</Aside>

## Instalación

1. Instala el paquete usando el siguiente comando:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/mdx' />

2. La configuración de StudioCMS debería incluir `@studiocms/mdx`:

   ```ts twoslash title="studiocms.config.mjs" {2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import mdx from '@studiocms/mdx';

    export default defineStudioCMSConfig({
      plugins: [
        mdx(),
      ],
    });
   ```
