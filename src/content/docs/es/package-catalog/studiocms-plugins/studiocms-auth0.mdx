Este plugin integra Auth0 como un proveedor de autenticación OAuth para StudioCMS, permitiendo la autenticación mediante Auth0. Configura la configuración necesaria, incluyendo las variables de entorno requeridas y las rutas de los endpoints.

## Uso

Agrega este plugin en tu configuración de StudioCMS. (`studiocms.config.mjs`)

```ts title="studiocms.config.mjs"
import { defineStudioCMSConfig } from 'studiocms/config';
import auth0 from '@studiocms/auth0';

export default defineStudioCMSConfig({
    // otras opciones aquí
    plugins: [auth0()]
});
```

## Variables de entorno requeridas

```sh title=".env"
CMS_AUTH0_CLIENT_ID=<your-client-id>
CMS_AUTH0_CLIENT_SECRET=<your-client-secret>
CMS_AUTH0_DOMAIN=<your-domain>
CMS_AUTH0_REDIRECT_URI=<your-redirect-uri>
```
