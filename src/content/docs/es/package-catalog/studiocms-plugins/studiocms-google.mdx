Este plugin integra la autenticación OAuth de Google en StudioCMS. Define la configuración necesaria, incluyendo las variables de entorno requeridas, los detalles del proveedor OAuth y las rutas de los endpoints para la autenticación.

## Uso

Agrega este plugin en tu configuración de StudioCMS. (`studiocms.config.mjs`)

```ts title="studiocms.config.mjs"
import { defineStudioCMSConfig } from 'studiocms/config';
import google from '@studiocms/google';

export default defineStudioCMSConfig({
    // other options here
    plugins: [google()]
});
```

## Variables de entorno requeridas

```sh title=".env"
CMS_GOOGLE_CLIENT_ID=<your-client-id>
CMS_GOOGLE_CLIENT_SECRET=<your-client-secret>
CMS_GOOGLE_REDIRECT_URI=<your-redirect-uri>
```
