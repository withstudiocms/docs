# Introducción

¡Este complemento agrega un editor WYSIWYG a tu proyecto StudioCMS con facilidad!

## Instalación

1. Instala el paquete usando el siguiente comando:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/wysiwyg' />

2. Tu configuración de StudioCMS ahora debe incluir `@studiocms/wysiwyg`:

   ```ts twoslash title="studiocms.config.mjs" {2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import wysiwyg from '@studiocms/wysiwyg';

    export default defineStudioCMSConfig({
      plugins: [
        wysiwyg(),
      ],
    });
   ```
