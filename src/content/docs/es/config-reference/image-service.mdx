Referencia del esquema de opciones de configuración de la integración de StudioCMS

```ts twoslash title="studiocms.config.mjs"
import { defineStudioCMSConfig } from 'studiocms/config';
// ---cut---
export default defineStudioCMSConfig({
  imageService: {
    cdnPlugin: 'cloudinary-js'
  },
});
```

## `cdnPlugin`

`cdnPlugin` es una enumeración que se utiliza para determinar qué plugin CDN usar para las imágenes, si está habilitado.

- **Tipo:** `'cloudinary-js'` | `undefined`
- **Valor predeterminado:** `undefined`
