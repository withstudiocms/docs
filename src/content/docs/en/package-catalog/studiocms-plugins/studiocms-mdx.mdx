# Introduction

This plugin enables MDX Support within StudioCMS.

## Installation

1. Install the package using the following command:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/mdx' />

2. Your StudioCMS config should now include `@studiocms/mdx`:

   ```ts twoslash title="studiocms.config.mjs" ins={2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import mdx from '@studiocms/mdx';

    export default defineStudioCMSConfig({
      plugins: [
        mdx(),
      ],
    });
   ```
