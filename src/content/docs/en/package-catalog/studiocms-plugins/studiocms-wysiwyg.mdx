# Introduction

This plugin adds a WYSIWYG editor to your StudioCMS project with ease!

## Installation

1. Install the package using the following command:

   <PackageManagers pkg="@studiocms/wysiwyg" />

2. Add `@studiocms/wysiwyg` to your Astro config file:

   ```ts twoslash title="studiocms.config.mjs" ins={2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import wysiwyg from '@studiocms/wysiwyg';

    export default defineStudioCMSConfig({
      plugins: [
        wysiwyg(),
      ],
    });
   ```
