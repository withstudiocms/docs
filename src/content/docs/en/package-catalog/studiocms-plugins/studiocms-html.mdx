# Introduction

This plugin enables HTML Support within StudioCMS.

## Installation

1. Install the package using the following command:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/html' />

2. Your StudioCMS config should now include `@studiocms/html`:

   ```ts twoslash title="studiocms.config.mjs" {2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import html from '@studiocms/html';

    export default defineStudioCMSConfig({
      plugins: [
        html(),
      ],
    });
   ```
