# Introduction

This plugin enables MD Support within StudioCMS.

## Installation

1. Install the package using the following command:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/md' />

2. Your StudioCMS config should now include `@studiocms/md`:

   ```ts twoslash title="studiocms.config.mjs" {2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import md from '@studiocms/md';

    export default defineStudioCMSConfig({
      plugins: [
        md(),
      ],
    });
   ```

## Options

### `studiocms` Flavored Markdown

```ts twoslash
const mdOptions = {
  flavor: 'studiocms',
  callouts: 'obsidian',
  autoLinkHeadings: true,
  discordSubtext: true,
  sanitize: {
    allowElements: [],
    blockElements: [],
    dropElements: [],
    allowAttributes: {},
    dropAttributes: {},
    allowComponents: false,
    allowCustomElements: false,
    allowComments: false,
  }
}
```
