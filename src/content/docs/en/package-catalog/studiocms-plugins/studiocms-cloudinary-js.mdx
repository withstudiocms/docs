# Introduction

This plugin enables the Cloudinary Image Service for StudioCMS

## Installation

1. Install the package using the following command:

   <PackageManagers pkg="studiocms" type='run' args='add @studiocms/cloudinary-image-service' />

2. Your StudioCMS config should now include `@studiocms/cloudinary-image-service`:

   ```ts twoslash title="studiocms.config.mjs" {2, 6}
   import { defineStudioCMSConfig } from 'studiocms/config';
   import cloudinaryJS from '@studiocms/cloudinary-image-service';

    export default defineStudioCMSConfig({
      plugins: [
        cloudinaryJS(),
      ],
    });
   ```

3. Set the required Environment Variables

   ```dotenv title=".env"
   CMS_CLOUDINARY_CLOUDNAME="demo"
   ```

## Usage

To enable the Image Service, you need to set the `preferredImageService` setting to `'cloudinary-js'`.

```ts twoslash title="studiocms.config.mjs" {2, 6}
import { defineStudioCMSConfig } from 'studiocms/config';
import cloudinaryJS from '@studiocms/cloudinary-image-service';

export default defineStudioCMSConfig({
  features: {
    preferredImageService: 'cloudinary-js'
  },
  plugins: [
    cloudinaryJS(),
  ],
});
```
