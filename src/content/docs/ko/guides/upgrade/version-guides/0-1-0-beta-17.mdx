최신 버전에서는 눈에 띄는 큰 변화는 없지만, StudioCMS의 전반적인 성능과 안정성을 향상시키기 위해 몇 가지 최적화와 버그 수정이 이루어졌습니다.

## 패치 개요

- **새로운 기능**
  - 대시보드에 새로운 피드백 시스템을 도입했습니다.
- **버그 수정**
  - Astro Fonts 관련 문제를 해결했습니다.
  - [`crypto gen-jwt`](/ko/how-it-works/cli/#crypto-gen-jwt) CLI 명령의 키 파일 경로 감지 기능을 개선했습니다.
- **개선 사항**
  - 더 나은 성능을 위해 [lodash](https://www.npmjs.com/package/lodash)를 [deepmerge-ts](https://www.npmjs.com/package/deepmerge-ts)로 대체했습니다.
  - 미들웨어의 Promise 처리 방식을 최적화하고 권한 수준 처리 방식을 업데이트했습니다.

<Aside title="참고">
`beta.16` 버전에서 비밀번호 해싱 알고리즘에 호환성이 깨지는 변경 사항이 있었습니다. 이전 비밀번호도 계속 작동은 하지만, `beta.17`로 업그레이드한 후에는 비밀번호를 업데이트하는 것을 권장합니다. 이전 버전의 비밀번호 해싱 지원은 향후 릴리스에서 제거될 예정입니다.
</Aside>
